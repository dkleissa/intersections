# intersections

Generate stylized maps of horrible intersections in your city.

## Demo

Check out the demo: ...

## Usage

    $ ./main.py [city [count]]

For example,

    $ ./main.py Boston 123

will generate 123 images of intersections in Boston.

    $ ./main.py Boston

will default 100. 

    $ ./main.py

with NO arguments will default to Baltimore.
